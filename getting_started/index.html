    Getting Started¶

    Welcome to the Bioengineering Research Group Lab Guide! This section will help you set up your machine for bioinformatics work. Follow these steps to install the necessary software and configure your environment.

    Installing UNIX/Linux¶

    For Windows Users: Using WSL2¶

    Windows Subsystem for Linux (WSL2) allows you to run a Linux distribution directly on Windows. Follow these steps to install WSL2:

    1. Enable WSL2:
    2. Open PowerShell as Administrator and run: powershell wsl --install

    3. Restart your computer if prompted.

    4. Install a Linux Distribution:

    5. Open Microsoft Store, search for "Ubuntu", and install it.
    6. Launch Ubuntu from the Start menu and follow the prompts to set up your Linux user.

    For Mac Users¶

    MacOS is UNIX-based, so you don't need to install anything additional. Open the Terminal application and you're ready to go.

    Installing VS Code¶

    Visual Studio Code (VS Code) is a powerful code editor with great support for development and bioinformatics.

    1. Download and Install VS Code:
    2. Go to Visual Studio Code.

    3. Download the installer for your operating system and follow the installation instructions.

    4. Install Recommended Extensions:

    5. Launch VS Code.
    6. Go to the Extensions view by clicking the square icon on the sidebar or pressing Ctrl+Shift+X.
    7. Search for and install the following extensions:
      • Python
      • Remote - WSL (for Windows users)
      • GitHub Copilot (if you have access)

    Create a GitHub Account¶

    GitHub is essential for version control and collaboration.

    1. Sign Up for GitHub:
    2. Go to GitHub.

    3. Click on "Sign up" and follow the instructions to create your account.

    4. Set Up Git on Your Machine:

    5. Open your terminal (Ubuntu on WSL2 for Windows or Terminal on Mac).
    6. Configure your Git username and email: bash git config --global user.name "Your Name" git config --global user.email "your.email@example.com"

    Get a GitHub Copilot Account¶

    GitHub Copilot is an AI-powered code completion tool.

    1. Request Access or Subscribe:
    2. Go to GitHub Copilot.

    3. Follow the instructions to request access or subscribe if itâ€™s generally available.

    4. Install GitHub Copilot Extension:

    5. Open VS Code.
    6. Go to the Extensions view (Ctrl+Shift+X).
    7. Search for "GitHub Copilot" and install the extension.

    Install Mamba/Miniforge¶

    Mamba is a fast package manager, and Miniforge is a minimal installer for conda.

    1. Install Miniforge:
    2. Download the Miniforge installer for your operating system from Miniforge GitHub Releases.

    3. Run the installer:

      • For Windows (in Ubuntu WSL2): bash wget <URL-to-Miniforge-installer> bash Miniforge3-Linux-x86_64.sh
      • For Mac: bash curl -L -O <URL-to-Miniforge-installer> bash Miniforge3-MacOSX-x86_64.sh

    4. Install Mamba:

    5. After installing Miniforge, open a new terminal session and run: bash conda install mamba -n base -c conda-forge

    Setting Up SSH Keys¶

    SSH keys allow secure connections to GitHub without needing to enter your password each time.

    1. Generate SSH Keys:
    2. Open your terminal.

    3. Run the following command and follow the prompts: bash ssh-keygen -t ed25519 -C "your.email@example.com"

    4. Add SSH Key to the SSH Agent:

    5. Start the ssh-agent: bash eval "$(ssh-agent -s)"

    6. Add your SSH key to the agent: bash ssh-add ~/.ssh/id_ed25519

    7. Add SSH Key to Your GitHub Account:

    8. Copy your SSH key to the clipboard: bash cat ~/.ssh/id_ed25519.pub | pbcopy
    9. Go to GitHub SSH Keys Settings.
    10. Click on "New SSH key", paste your key, and save.

    Congratulations! Your machine is now set up for bioinformatics work. Proceed to the specific sections on Genome Assembly, Annotation, and Mining to start using the tools and protocols in our lab.
